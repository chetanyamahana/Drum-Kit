# Drum-Kit
This is an interactive web application that allows you to play drum sounds using your mouse or keyboard. Built with HTML, CSS, and JavaScript, the Drum Kit features a virtual drum set with different drum pads representing various percussion instruments.</br>
When you click on a drum pad or press the corresponding key on your keyboard, the respective drum sound will play. The website provides a fun and engaging way to experience playing a drum set right from your web browser.</br>
Key Features:</br>
</br>
->Virtual drum pads for different drum sounds.</br>
->Play drums with mouse clicks or keyboard presses.</br>
->Responsive design for optimal experience on various devices.</br>
